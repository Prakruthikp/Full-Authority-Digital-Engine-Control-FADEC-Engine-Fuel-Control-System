Full Authority Digital Engine Control (FADEC) & Engine Fuel Control System
Project Overview

This repository presents the design and implementation of a complete Engine Fuel Control System (FCS) using MATLAB/Simulink for turbojet engine applications. The project models the core functions of a modern Full Authority Digital Engine Control (FADEC), including fuel scheduling, engine start logic, closed-loop control, sensor processing, actuator management, and safety monitoring.

Project Objectives

Develop a complete FADEC architecture
Design engine fuel control logic
Implement fuel scheduling algorithms
Develop engine start and shutdown sequences
Integrate engine sensors
Control fuel metering valve (FMV)
Monitor engine operating conditions
Implement safety and protection logic
Validate closed-loop engine performance
Simulate engine operating scenarios
